Zuletzt aktualisiert: Juni 2026

Kurzfassung: Emilia sammelt keine persönlichen Daten. Kein Konto, keine Registrierung, keine E-Mail-Adresse. Alle Daten bleiben ausschließlich auf deinem Gerät.

Ivan Janjić

2. Welche Daten werden gesammelt?

Keine. Emilia benötigt weder ein Benutzerkonto noch eine Registrierung. Es werden keinerlei persönlichen Daten erhoben, gespeichert oder übertragen.

3. Kalender-Zugriff

Emilia fragt nach Zugriff auf deinen iOS-Kalender, um:

  • Deine Termine anzuzeigen
  • Neue Termine direkt im iOS-Kalender zu erstellen
  • Geteilte Kalender anderer Personen in der Familie-Ansicht anzuzeigen

Die Kalenderdaten verlassen dein Gerät zu keinem Zeitpunkt. Sie werden ausschließlich lokal verarbeitet und für das Widget über eine App Group (verschlüsselt auf dem Gerät) bereitgestellt.

4. Widget

Das Emilia-Widget liest Termindaten aus einem verschlüsselten Bereich auf deinem Gerät (App Group). Diese Daten werden nicht ins Internet übertragen.

5. Drittanbieter

Emilia verwendet keine Analyse-Tools, keine Werbenetzwerke und keine Tracking-Dienste. Es werden keine Daten an Dritte weitergegeben.

6. Kinder

Da keine personenbezogenen Daten gesammelt werden, ist Emilia für alle Altersgruppen geeignet.

7. Änderungen

Diese Datenschutzerklärung kann gelegentlich aktualisiert werden. Die jeweils aktuelle Version ist immer auf dieser Seite verfügbar.
